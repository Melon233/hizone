# Hizone
## 需求
1. 注册/登录
2. 用户信息查看/编辑
3. 关注/取消关注/删除粉丝
4. 发布/删除帖子
5. 内容点赞/收藏/评论/回复的增删查
6. 帖子/用户关键词搜索
7. 获取推送
8. 浏览帖子

## 环境依赖
1. 所有pom指定的内部依赖
2. JDK 21
3. MySQL 8.0.40
4. Redis 5.0.14.1
5. Nacos 2.4.3
6. Seata 2.1.0
7. Sentinel 1.8.8
8. Elasticsearch 8.17.3
9. RabbitMQ 4.0
10. Erlang 15.2.4
11. Nginx 1.26.3
